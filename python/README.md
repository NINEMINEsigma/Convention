# Convention

Convention是一个综合性的Python工具库，提供了多个实用模块：

## 功能模块

- Algorithm: 算法工具
- Audio: 音频处理工具
- DataBase: 数据库操作工具
- File: 文件操作工具
- Lang: 语言处理工具
- LLM: 大语言模型工具
- MathEx: 数学扩展工具
- ML: 机器学习工具
- Str: 字符串处理工具
- UI: 用户界面工具
- Visual: 视觉处理工具
- Web: Web相关工具

## 安装与构建

使用python构建:

```bash
python -m build
```

使用pip安装：

```bash
pip install dist/Convention-version.tar.gz
```

或者

```bash
pip install dist/Convention-version.whl
```

## 生成依赖文件

```python
import Convention
ReleaseFailed2Requirements()
```

## 使用示例

```python
from Convention import Str
from Convention import Web
```
